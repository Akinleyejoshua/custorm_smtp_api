API Documentation

AetherSMTP runs a local Node.js engine providing both an SMTP port for legacy integration (e.g. Laravel, XAMPP, PHP `mail()`) and a robust REST API for modern microservices.

1. SMTP server integration

Configure your application or framework to route mail through this custom server:

{
  "SMTP_HOST": "127.0.0.1",
  "SMTP_PORT": 2525,
  "SMTP_SECURE": false,
  "SMTP_USER": "",
  "SMTP_PASS": ""
}

2. REST API: Send Bulk Emails

POST
/api/send

Use this endpoint to trigger email sends (supports bulk queues).

Request Headers

Content-Type: application/json

Request Body Payload

{
  "to": "user1@example.com, user2@example.com", // String or Array
  "from": "app-notifications@mydomain.com",
  "subject": "System Warning: Storage full",
  "html": "<h1>Storage Capacity Alert</h1><p>Please free up disk space...</p>",
  "simulate": true // set to false for real MX delivery (DNS resolution)
}

Response Format (Success 200 OK)

{
  "success": true,
  "summary": {
    "total": 2,
    "success": 2,
    "failed": 0
  },
  "results": [
    {
      "id": "snd_34f89d",
      "type": "sent",
      "timestamp": "2026-06-17T01:30:00.000Z",
      "from": "app-notifications@mydomain.com",
      "to": "user1@example.com",
      "subject": "System Warning: Storage full",
      "status": "success",
      "log": "Simulation mode active...",
      "mx": "smtp.simulated.local"
    }
  ]
}
